# Project 2 - Matrix Operations Toolkit

## 📌 Project Overview

This project was developed as part of my **QSkill Python Development Internship**.

The original internship task was to build a **Python terminal application** that performs various matrix operations. After successfully completing the required console application, I extended the project by developing an additional **Flask-based web application** to provide the same functionality through a modern graphical user interface.

This project demonstrates both **Python programming fundamentals** and **basic full-stack web development**.

---

# 📁 Project Contents

This project contains two implementations:

## 1️⃣ Python Terminal Application (Internship Requirement)

The console application is the original internship project.

### Features

* Matrix Addition
* Matrix Subtraction
* Matrix Multiplication
* Matrix Transpose
* Matrix Determinant
* Input Validation
* Error Handling
* Menu Driven Interface

Main File

```
matrix_calculator.py
```

---

## 2️⃣ Flask Web Application (Additional Enhancement)

To improve the project beyond the internship requirements, I developed a complete web application using Flask.

The web version provides the same matrix operations through an interactive and responsive user interface.

### Features

# ▶️ Running the Python Terminal Application

1. Open the project folder.
2. Open a terminal.
3. Run:

```
python matrix_calculator.py
```

---

# ▶️ Running the Flask Web Application

1. Open the **Web_Application** folder.

2. Install the required packages.

```
pip install -r requirements.txt
```

3. Start the Flask server.

```
python app.py
```

4. Open your browser.

```
http://127.0.0.1:5000
```
